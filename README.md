# qt-snake-vim-train

A project for my school final exam.

Ofc u can train vim motions with this snake game.

## Install

```zsh
python -m venv venv &&
source venv/bin/activate &&
pip install -r requirements.txt &&
python main.py
```

## Usage

During the game:

- press `h` to go left
- press `l` to go right
- press `k` to go up
- press `j` to go down
- press `r` to restart
- press `space` to toggle speed up
- press `q` to quit
